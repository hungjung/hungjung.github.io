PHPStorm開發工具使用

初始化設定

  • Git 執行指令的設定的確認
    1. File / Settings
    2. settings面板左側的「Version Control」/「Git」
    3. settings面板右側確認Git執行檔是否載入
  • 設定程式碼字型
    1. File / Settings
    2. settings面板左側的「Editor」/「Font」
    3. settings面板右側確認字型的設定
  • 設定程式碼配色風格

安裝外掛

  • .ignore
  • .env files support
  • PHP composer.json support
  • Laravel plugin
    • phpstorm 運用 PHPDoc
  • Bootstrap 3

快速鍵

  • 已知檔名開始檔案 ⇒ ctrl + shift + n
  • 擴增選取文字(expand selection) ⇒ ctrl + w
  • 查看程式碼前後差異(git diff) ⇒ ctrl + d
  • 整理程式碼(reformat code) ⇒ ctrl + alt + l
